Core Capabilities

Everything your support team needs

Built from the ground up for customer-facing video support — not stitched together from generic SDKs.

📋

Session Management

Agents create sessions instantly. Customers join via a secure token link — no account needed, no app to install.

🎥

Server-Routed Video

All audio and video routes through your own SFU server. No P2P, no third-party media infrastructure, full data sovereignty.

💬

In-Call Chat

Real-time messaging during the call. Every message is persisted to the session record and retrievable after the call ends.

🔴

Call Recording

Agents start and stop recording at any time. Once processed, the file is available for download with status tracking.

📎

File Sharing

Share images, PDFs, and documents directly in the chat. Files are stored securely and linked to the session record.

📊

Admin Dashboard

Live session overview, participant details, event logs, and the ability to terminate any session — built for ops teams.

Access Control

Two roles, clearly enforced

🎧 Call Agent

Full session control. The agent drives every session from creation to close.

✓
Create and end sessions
✓
Invite customers via link/token
✓
Start and stop recording
✓
Mute/video toggle for self
✓
Access session history & records
👤 Customer

Read-only session access. Customers join and participate — nothing more.

✓
Join via invite link or token
✓
See and hear the agent
✓
Toggle own audio/video
✓
Send and receive chat messages
✗
Cannot create or end sessions
